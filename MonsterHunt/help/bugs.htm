Bugs

Here are some bugs that you should be aware of:

- Only works with v436 and up

- The Monster Madness mutator may cause some problems in certain maps, but this is only if a GiantGasbag is used as the boss in a MA game or if a GiantGasbag is set to trigger an event. No maps that are distributed with MH have conflicts in this regard but it's something to keep in mind.

- Bot's are not as smart as they could be, they just run off doing their own thing. The best thing to do in a game with bots, is to order them to cover you and they'll follow you and help out. Bots only push buttons/pull levers etc. very seldom.

If you can find any bugs or problems(no matter how small), please let me know.
